# Weather-react-app

The API is called twice, once for current real-time weather conditions and once for the next 7 day weather forecast
The forecast shows possible minimum and maximum temperature
API is called in App.jsx, it includes 3 components: search bar, current-weather, and forecast

# To run the project locally

Make sure you have Node.js installed along with npm/npx command

- Clone the project
- Go to Project Directory in Terminal
- Run the following commands in order:
  `npm install`
  `npm run dev`
- Open the localhost link available in Terminal
